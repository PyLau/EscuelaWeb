Welcome to Lau's GitHub Page.

Hi! I'm Laura and this is my GitHub page. Aquí voy a colocar un resumen muy corto de los cursos que vi en las clases en Escuela Web. En mi GitHub se encuentra el repositorio EscuelaWeb donde está todo lo que hice y a continuación en cada resumen de la clase anexo un link con el código de esa lección y mi respuesta a la tarea.

Diseño Web HTML5 y CSS3 desde cero.

Herramienta: Editor de texto como SublimeText 2 o 3, también Brackets o Atom. En el blog del profesor: http://josecastro.com.ve/sublime-text-3/ están los pasos para instalar los plugins de SublimeText3 que estaremos utilizando a través del curso.

1. Introducción a la maquetación HTML/CSS

En la primera clase vimos un poco de teoria muy básica de HTML5 y CSS3, luego pusimos en práctica algunas etiquetas en una página html, sin estilos: http://pylau.github.io/EscuelaWeb/HTML5yCSS3/lauraparra/index.html Luego, en otro html agregamos una hoja de estilo y le definimos estilos a varios elementos.
https://pylau.github.io/EscuelaWeb/HTML5yCSS3/lauraparra/day1.html

Homework

En la página web CSS Zen Garden está disponible para su descarga el archivo HTML de dicha página, donde muchas personas publican su archivo .css para visualizar el cambio de estilo de ella. Para la tarea descargamos el HTML y cada uno le agrego el estilo de su preferencia a traves de una hoja de estilos.
